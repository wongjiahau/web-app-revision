## Headings
Used to create headings. `<h1>` is the biggest and `<h6>` is the smallest.
```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

## Paragraph
- `<p>` is used to identify blocks of paragraph text.  
- `<br>` is linebreak
```html
<p>This is a paragraph</p>
<br/>
```
